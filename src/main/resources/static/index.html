About JSON Epoch Converter

A free online tool to convert Unix/epoch timestamps embedded inside JSON data into human-readable dates. Paste your JSON, upload a file, or drop it onto the panel — epoch fields are auto-detected and converted instantly. No sign-up, no limits.

Your Sensitive Data Is Safe

Many JSON payloads contain sensitive information — user IDs, auth tokens, PII, or business-critical data. Unlike most online tools, JSON Epoch Converter processes everything entirely inside your browser. Your JSON is never uploaded, transmitted, or stored anywhere.

Zero Data Transmission

When you click Transform, no network request is made. The entire conversion runs in JavaScript inside your browser tab. Verify it yourself — open DevTools → Network and watch zero API calls fire.

No Server Storage

Nothing is logged, cached, or persisted. Even a 50 MB JSON file stays in your browser's memory and is discarded when you close the tab.

Open to Inspect

Built with plain HTML, CSS, and JavaScript. Open DevTools and read the source — no hidden trackers or data-exfiltration scripts anywhere near your JSON.

How It Works

The tool parses your JSON in the browser and walks every field recursively. For each numeric value, it checks the digit count to determine the precision unit — 10 digits means seconds, 13 means milliseconds, 16 means microseconds, and 19 means nanoseconds. Values outside the valid epoch range are left untouched.

The value is then converted using the browser's built-in Intl.DateTimeFormat API with your chosen timezone and pattern. For 19-digit nanosecond timestamps, JavaScript's BigInt is used to preserve precision.
