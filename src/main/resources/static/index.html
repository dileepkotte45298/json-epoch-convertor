JSON Epoch Converter

A free online tool to convert Unix/epoch timestamps embedded inside JSON data into human-readable dates. Paste your JSON, upload a file, or drop it onto the panel — epoch fields are auto-detected and converted instantly.

Seconds & Milliseconds

Automatically detects 10-digit (seconds) and 13-digit (milliseconds) Unix timestamps in any JSON structure.

Microseconds & Nanoseconds

Full support for high-precision 16-digit (microseconds) and 19-digit (nanoseconds) epoch values used in modern distributed systems.

Any Timezone

Convert epoch timestamps to any timezone worldwide. Search from 600+ IANA timezones including UTC, IST, PST, EST, and more.

Custom Date Formats

Choose from presets (ISO 8601, US, EU) or define your own date format pattern using standard Java date format tokens.

How to Use
  1. Paste or Upload JSON
    Paste your JSON directly into the Input JSON panel, or click Upload JSON to load a .json file. You can also drag and drop a file onto the panel.
  2. Transform
    Click Transform. Epoch fields are auto-detected and converted automatically — no manual selection needed.
  3. Select Fields (optional)
    Use Auto-detect fields to review which fields will be converted, or type a custom path (e.g. data.ts) and click + Add to target specific fields.
  4. Choose Timezone & Format
    Search and select your Timezone. Pick a Date Format preset (Default, US, EU, ISO 8601) or type a custom pattern.
  5. View & Export Output
    The converted JSON appears in the Code panel. Use Copy JSON to copy it to clipboard, or Download to save it as a file.
